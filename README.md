# 中文 · SRS — Spaced Repetition Review for Chinese

A standalone web app that extracts vocabulary, grammar patterns, and useful phrases from the [Learning-Chinese](https://github.com/davidbcn07/Learning-Chinese) exercise vault and presents them as flashcards using the SM-2 spaced repetition algorithm (the same algorithm behind Anki).

## Quick Start

```bash
# Install dependencies
uv sync

# Import data from the Learning-Chinese directory
uv run import_data.py

# Start the server
uv run app.py
```

Then open **http://127.0.0.1:5000** in your browser.

> The Learning-Chinese repo is expected at `../Learning-Chinese` by default (relative to CWD). See [Configuration](#configuration) for other setups.

## Features

- **1,800+ cards** automatically extracted from 79 exercise files
- **SM-2 Spaced Repetition** — the same algorithm Anki uses
- **Three card types**: Vocabulary (pinyin + meaning), Grammar patterns (with examples), Useful phrases
- **Incremental import** — re-running `import_data.py` adds new cards and updates content without losing your SRS progress
- **Responsive UI** — works on desktop and mobile
- **Keyboard shortcuts** — tap/space to flip, 0-5 to rate
- **Browse & search** — filter by type and search across all cards

## Card Types

| Type | Front | Back |
|------|-------|------|
| Vocab | Chinese character(s) | Pinyin + English meaning |
| Grammar | Pattern name | Structure rules + examples |
| Phrase | Chinese phrase | English translation |

## SM-2 Rating Guide

When reviewing a card, rate how well you remembered it on a scale of 0–5:

| Rating | Meaning | Effect |
|--------|---------|--------|
| 0 | Complete blackout | Reset to 1 day |
| 1 | Wrong, but remembered after seeing answer | Reset to 1 day |
| 2 | Wrong, but felt familiar | Reset to 1 day |
| 3 | Correct with difficulty | Interval × EF (small increase) |
| 4 | Correct after some hesitation | Interval × EF (moderate increase) |
| 5 | Perfect, immediate recall | Interval × EF (large increase) |

Failed cards (rating 0–2) reset to step 1 and are re‑queued for the next day.

## Configuration

The import script and server accept the paths through multiple channels (priority order: CLI arg → env var → default):

### Import Script

```bash
# CLI argument
uv run import_data.py /path/to/Learning-Chinese

# Environment variable
LEARNING_CHINESE_DIR=/path/to/Learning-Chinese uv run import_data.py

# Custom database path
uv run import_data.py /path/to/Learning-Chinese /path/to/custom.db
```

### Server

```bash
# Custom database path
SRS_DB_PATH=/path/to/custom.db uv run app.py
```

## API Endpoints

| Method | Path | Description |
|--------|------|-------------|
| `GET` | `/` | Frontend HTML page |
| `GET` | `/api/stats` | Dashboard statistics |
| `GET` | `/api/review/next` | Next due card(s) |
| `GET` | `/api/review/new` | New (unreviewed) cards |
| `POST` | `/api/review` | Submit a review rating |
| `POST` | `/api/review/session` | Batch submit multiple reviews |
| `GET` | `/api/cards` | Browse/search cards |
| `GET` | `/api/card/<id>` | Single card detail |
| `GET` | `/api/categories` | List all categories |
| `GET` | `/api/themes` | List all exercise themes |
| `GET` | `/api/activity` | Review activity (30 days) |
| `POST` | `/api/reset` | Reset a card to unreviewed |

## Updating After New Exercises

Whenever new exercises are added to the Learning-Chinese repo, simply re‑run the import:

```bash
uv run import_data.py
```

Only new cards are added. Existing cards have their content fields (meaning, pinyin, theme) refreshed, but **SRS progress (easiness, interval, repetitions, next review date) is always preserved**.
